# License Plate Extraction with YOLOv10 and PaddleOCR & Save Data to SQL Database

## How to run:

```bash
git clone https://github.com/THU-MIG/yolov10.git
```

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
```

```bash
pip install -r requirements.txt
```

```bash
cd yolov10
```

```bash
pip install -e .
```

```bash
cd ..
```

```bash
python sqldb.py
```

```bash
python main.py
